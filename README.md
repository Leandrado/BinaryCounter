Binary Counter


**About the Project**

This project is a binary counter composed of three main elements: the decimal number input, the bit display, and the control buttons.

It allows users to convert numbers between decimal (base 10) and binary (base 2), as well as manually manipulate individual bits.


**How it works**

The project is built using JavaScript functions responsible for converting numbers between different bases.

Users can:

-Enter a decimal number and see its binary representation

-Modify the binary bits and see the decimal value update 
automatically

-Change the number of bit groups displayed on the screen

**How to use?**

-The top input field is used to enter a decimal (base 10) number, which is automatically converted to binary.

-The groups of 4 bits represent the binary number. You can click the bits to change their values and update the decimal number.

-The +1 button increases the decimal number by 1 and updates the binary value.

-The −1 button decreases the decimal number by 1 and updates the binary value.

-The hidden input at the top-right allows you to change the number of 4-bit groups (from 1 to 6).
